# Mantine v8 UI Guidance

Leetgrind uses Mantine v8 for application UI, shared primitives, styling, color
scheme handling, and accessible component behavior.

When asking AI tools about Mantine in this repository:

1. Mention that the project uses Mantine v8.
2. Reference the Mantine LLM documentation URL: `https://mantine.dev/llms.txt`.
3. Prefer Mantine components and style props over Tailwind-style utility
   classes or one-off hand-built UI primitives.

Local reference snapshot:

- `docs/vendor/mantine-v8-llms.txt`

The local snapshot is an LLM-oriented index of Mantine documentation pages. Use
the official URL when checking current component behavior, and treat the local
snapshot as a convenient repo-level reminder of available documentation areas.
